# Book Database Node.js Project

Цей проєкт дозволяє керувати базою даних книг за допомогою **Node.js** та **PostgreSQL**.  
Можна додавати книги, переглядати список, оновлювати рейтинг та видаляти записи.

---

## 📂 Файли проєкту

- `database.js` – головний файл з усіма функціями CRUD для таблиці `book`  
- `.env` – конфігураційний файл з URL підключення до бази  
- `package.json` – налаштування проєкту для Node.js  
- `server.js` – (може містити серверну логіку, не обов’язково)  
- `img/` – картинки (якщо потрібно)  

---

## ⚙️ Налаштування

1. Встановіть Node.js (версія >= 18)
2. Встановіть PostgreSQL або скористайтеся онлайн базою (Neon, Supabase)  
3. Створіть базу даних, наприклад `books`:

```sql
CREATE DATABASE books;
